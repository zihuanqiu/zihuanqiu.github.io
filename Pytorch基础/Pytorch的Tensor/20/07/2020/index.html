Pytorchзҡ„Tensor

еҸ‘иЎЁдәҺ 20-07-2020 жӣҙж–°дәҺ 02-02-2023 еҲҶзұ»дәҺ Pytorch

1. еј йҮҸжҳҜд»Җд№Ҳ

еј йҮҸжҳҜдёҖдёӘеӨҡз»ҙж•°з»„пјҢе®ғжҳҜж ҮйҮҸгҖҒеҗ‘йҮҸгҖҒзҹ©йҳөзҡ„й«ҳз»ҙжӢ“еұ•

Pytorch0.4.0зүҲејҖе§ӢпјҢVariableе№¶е…ҘTensor

  • dtype:еј йҮҸзҡ„ж•°жҚ®зұ»еһӢпјҢеҰӮtorch.FloatTensor,torch.cuda.FloatTensor

  • shape:еј йҮҸзҡ„еҪўзҠ¶пјҢеҰӮ(64,3,224,224)

  • device:еј йҮҸжүҖеңЁи®ҫеӨҮпјҢGPU/CPUпјҢжҳҜеҠ йҖҹзҡ„е…ій”®

2. еј йҮҸзҡ„еҲӣе»ә

2.1 д»ҺndarrayеҲӣе»ә

2.1.1 йҖҡиҝҮtorch.tensorеҲӣе»әеј йҮҸ

еҠҹиғҪпјҡд»ҺdataеҲӣе»әtensor

  • data: ж•°жҚ®, еҸҜд»ҘжҳҜlist, numpy

  • dtype : ж•°жҚ®зұ»еһӢпјҢй»ҳи®ӨдёҺdataзҡ„дёҖиҮҙ

  • device : жүҖеңЁи®ҫеӨҮ, cuda/cpu

  • requires_gradпјҡжҳҜеҗҰйңҖиҰҒжўҜеәҰ

  • pin_memoryпјҡжҳҜеҗҰеӯҳдәҺй”ҒйЎөеҶ…еӯҳ

2.1.2 йҖҡиҝҮtorch.from_numpyеҲӣе»әеј йҮҸ

еҠҹиғҪпјҡд»ҺnumpyеҲӣе»әtensor
жіЁж„ҸдәӢйЎ№пјҡд»Һtorch.from_numpyеҲӣе»әзҡ„tensorдәҺеҺҹndarrayе…ұдә«еҶ…еӯҳпјҢеҪ“дҝ®ж”№е…¶дёӯдёҖдёӘзҡ„ж•°жҚ®пјҢеҸҰеӨ–дёҖдёӘд№ҹе°Ҷдјҡиў«ж”№еҠЁ

2.2 дҫқжҚ®ж•°еҖјеҲӣе»ә

2.2.1 йҖҡиҝҮtorch.zerosеҲӣе»әеј йҮҸ

еҠҹиғҪпјҡдҫқsizeеҲӣе»әе…Ё0еј йҮҸ

  • size: еј йҮҸзҡ„еҪўзҠ¶, еҰӮ(3, 3)гҖҒ(3, 224,224)

  • out : иҫ“еҮәзҡ„еј йҮҸ

  • layout : еҶ…еӯҳдёӯеёғеұҖеҪўејҸ, жңүstrided,sparse_cooзӯү

  • device : жүҖеңЁи®ҫеӨҮ, gpu/cpu

  • requires_gradпјҡжҳҜеҗҰйңҖиҰҒжўҜеәҰ

дёӨз§Қиҫ“еҮәж–№ејҸпјҡзӣҙжҺҘеӨҚеҲ¶пјҢжҲ–иҖ…йҖҡиҝҮеҸӮж•°out

2.2.2 йҖҡиҝҮtorch.zeros_likeеҲӣе»әеј йҮҸ

еҠҹиғҪпјҡдҫқinputеҪўзҠ¶еҲӣе»әе…Ё0еј йҮҸ

  • intput: еҲӣе»әдёҺinputеҗҢеҪўзҠ¶зҡ„е…Ё0еј йҮҸ
  • dtype : ж•°жҚ®зұ»еһӢ
  • layout : еҶ…еӯҳдёӯеёғеұҖеҪўејҸ

2.2.3 йҖҡиҝҮtorch.onesеҲӣе»әеј йҮҸ

еҠҹиғҪпјҡдҫқinputеҪўзҠ¶еҲӣе»әе…Ё1еј йҮҸ

  • size: е®ҡд№үиҫ“еҮәеј йҮҸеҪўзҠ¶зҡ„ж•ҙж•°еәҸеҲ—гҖӮеҸҜд»ҘжҳҜеҸҜеҸҳж•°йҮҸзҡ„еҸӮж•°пјҢд№ҹеҸҜд»ҘжҳҜеҲ—иЎЁжҲ–е…ғз»„гҖӮ
  • dtype : ж•°жҚ®зұ»еһӢ
  • layout : еҶ…еӯҳдёӯеёғеұҖеҪўејҸ
  • device : жүҖеңЁи®ҫеӨҮ, gpu/cpu
  • requires_gradпјҡжҳҜеҗҰйңҖиҰҒжўҜеәҰ

2.2.4 йҖҡиҝҮtorch.ones_likeеҲӣе»әеј йҮҸ

  • input (Tensor) вҖ“ the size of input will determine size of the output tensor.
  • dtype (torch.dtype, optional) вҖ“ the desired data type of returned Tensor. Default: if None, defaults to the dtype of input.
  • layout (torch.layout, optional) вҖ“ the desired layout of returned tensor. Default: if None, defaults to the layout of input.
  • device (torch.device, optional) вҖ“ the desired device of returned tensor. Default: if None, defaults to the device of input.
  • requires_grad (bool, optional) вҖ“ If autograd should record operations on the returned tensor. Default: False.
  • memory_format (torch.memory_format, optional) вҖ“ the desired memory format of returned Tensor. Default: torch.preserve_format.

2.2.5 йҖҡиҝҮtorch.fullеҲӣе»әеј йҮҸ

  • size (intвҖҰ) вҖ“ a list, tuple, or torch.Size of integers defining the shape of the output tensor.
  • fill_value вҖ“ the number to fill the output tensor with.
  • out (Tensor, optional) вҖ“ the output tensor.
  • dtype (torch.dtype, optional) вҖ“ the desired data type of returned tensor. Default: if None, uses a global default (see torch.set_default_tensor_type()).
  • layout (torch.layout, optional) вҖ“ the desired layout of returned Tensor. Default: torch.strided.
  • device (torch.device, optional) вҖ“ the desired device of returned tensor. Default: if None, uses the current device for the default tensor type (see torch.set_default_tensor_type()). device will be the CPU for CPU tensor types and the current CUDA device for CUDA tensor types.
  • requires_grad (bool, optional) вҖ“ If autograd should record operations on the returned tensor. Default: False.

е®һйӘҢпјҡ

2.2.6 йҖҡиҝҮtorch.full_likeеҲӣе»әеј йҮҸ

еҠҹиғҪпјҡдҫқinputеҪўзҠ¶еҲӣе»әжҢҮе®ҡж•°жҚ®зҡ„еј йҮҸ

  • size: еј йҮҸзҡ„еҪўзҠ¶, еҰӮ(3, 3)
  • fill_value : еј йҮҸзҡ„еҖј

2.2.7 йҖҡиҝҮtorch.arangeеҲӣе»әзӯүе·®ж•°еҲ—еј йҮҸ

еҠҹиғҪпјҡеҲӣе»әзӯүе·®зҡ„1з»ҙеј йҮҸ

жіЁж„ҸдәӢйЎ№пјҡж•°еҖјеҢәй—ҙдёә[start, end)

  • start: ж•°еҲ—иө·е§ӢеҖј
  • end : ж•°еҲ—вҖңз»“жқҹеҖјвҖқ
  • step: ж•°еҲ—е…¬е·®пјҢй»ҳи®Өдёә1

2.2.8 йҖҡиҝҮtorch.linspaceеҲӣе»әеқҮеҲҶж•°еҲ—еј йҮҸ

еҠҹиғҪпјҡеҲӣе»әеқҮеҲҶзҡ„1з»ҙеј йҮҸ

жіЁж„ҸдәӢйЎ№пјҡж•°еҖјеҢәй—ҙдёә[start, end]

  • start: ж•°еҲ—иө·е§ӢеҖј
  • end : ж•°еҲ—з»“жқҹеҖј
  • steps: ж•°еҲ—й•ҝеәҰ

2.2.9 йҖҡиҝҮtorch.logspaceеҲӣе»әеҜ№ж•°еқҮеҲҶж•°еҲ—еј йҮҸ

еҠҹиғҪпјҡеҲӣе»әеҜ№ж•°еқҮеҲҶзҡ„1з»ҙеј йҮҸпјӣеә”иҜҘжҳҜеҸ–дәҶеҜ№ж•°д»ҘеҗҺжҳҜеқҮеҲҶзҡ„ж„ҸжҖқ

жіЁж„ҸдәӢйЎ№пјҡй•ҝеәҰдёәsteps, еә•дёәbase

  • start: ж•°еҲ—иө·е§ӢеҖј
  • end : ж•°еҲ—з»“жқҹеҖј
  • steps: ж•°еҲ—й•ҝеәҰ
  • base : еҜ№ж•°еҮҪж•°зҡ„еә•пјҢй»ҳи®Өдёә10

е…¶е®һе°ұжҳҜдёҖдёӘжҢҮж•°зӯүжҜ”ж•°еҲ—пјҢlogspaceеҜ№ж•°еқҮеҲҶзҡ„ж„ҸжҖқжҳҜпјҢеҸ–дәҶеҜ№ж•°д»ҘеҗҺжҳҜеқҮеҲҶ

2.2.10 йҖҡиҝҮtorch.eyeеҲӣе»әеҚ•дҪҚеҜ№и§’зҹ©йҳө

еҠҹиғҪпјҡеҲӣе»әеҚ•дҪҚеҜ№и§’зҹ©йҳөпјҲ 2з»ҙеј йҮҸпјү

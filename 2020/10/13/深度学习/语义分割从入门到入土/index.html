语义分割从入门到入土

FCN——开山之作

DeepLab系列

DeepLabV1——DeepLab系列之雏形

​ 论文题目：SEMANTIC IMAGE SEGMENTATION WITH DEEP CONVOLUTIONAL NETS AND FULLY C ONNECTED CRFs

提出了使用DCNNs（深度卷积网络）解决语义分割问题的两大挑战

  1. 信号下采样
  2. 空间不变性

1. 解决特征分辨率问题

​ 第一个问题是由于卷积神经网络重复使用卷积和池化的组合操作来降低特征的分辨率；第二个问题是由于DCNNs本身的结构属性（卷积池化都是具有空间不变的操作），限制了DCNNs对空间信息提取的能力。

​ 针对第一个问题，作者使用了空洞算法解决这个问题（也就是之后的空洞卷积，只不过这时候还没发展到给卷积核插0值，而是对输入特征做抽取）；空洞算法能扩大卷积的感受域，具体做法是对输入抽取，然后再做卷积。这样一个卷积核就能捕获原图像中更大范围的区域。

​ 上图是论文中对空洞算法在1维情况下的示例说明（在deeplabV2中我们会看到真正的空洞卷积，由Nobel identity的原理，对输入抽取等效于对卷积核做插值，变成空洞卷积核）

由Nobel identity的原理可以做一个形象的说明：

（a）可以视为池化之后卷积，那么与之等效的系统如（b）所示。H(zM)H(z^{M})H(zM)是H(z)H(z)H(z)通过插0值上采样MMM倍的结果，那么就可以等效为对卷积核做插0值，得到空洞卷积核。

​ 所以一个在池化后的卷积核被等效为了在池化前的空洞卷积核，因此使用空洞卷积可以避免过多地堆叠卷积池化。使用较少的卷积池化就可以获得相同的效果，避免了特征分辨率的过度下降。

​ 本文的模型也是从在ImageNet中预训练的VGG16微调得到的。与FCN的想法类似，把全连接层改为卷积层得到一个全卷积网络。但是FCN为了得到较大的感受域得到的特征图只相当于原输入图像尺寸1/32，本文作者认为这是过于粗糙的。因此把最后两层池化层去掉，并且使用了空洞算法，得到了1/8尺寸的特征图。并且作者认为他们得到的1/8特征图是非常平滑的，不需要使用反卷积上采样而是直接使用双线性插值直接上采样8倍得到原图像尺寸的特征图。并且抨击了FCN使用反卷积造成过多的训练参数和计算量（FCN训练了几天而他们只训练了10个小时）。

​ DCNN的训练使用交叉熵损失函数，对1/8特征图双线性插值上采样8倍的结果与GroundTrue做交叉熵损失，用SGD训练。

2. 解决空间不变性问题

​ 针对第二个问题，本文使用了全连接的条件随机场（CRF），对使用双线性插值8倍上采样的特征图做细节修饰。经过多轮CRF迭代最终得到语义分割的结果。全连接的CRF对特征图的任意两个像素点之间都建立了联系，每一个像素都是一个CRF节点，使用快速算法得到可以接受的计算效率。

​ 上图中第二列第二个图就是DCNN的输出，已经基本能看了。使用CRF后能获得IoU 4%的提升。

​ 另外，注意CRF只是对DCNN的输出做修饰（毕竟只有4%的提升），所以本文对1/8特征图直接双线性插值上采样8倍的操作已经能获得不错的分割结果。这是与FCN相比非常惊人的，因为FCN使用了反卷积。所以deeplabV1能达到这样的效果应该归功于空洞算法对特征图分辨率的保留。

​ CRF是在DCNN训练结束以后单独训练的。整个网络就DCNN+CRF的两端拼接结构。

3. 其他细节

​ 模型从ImageNet预训练的VGG16改造而来，替换全连接为全卷积。并且将最后的1000类分类器改为21类分类器（Pascal Voc为20类+背景）

​ 作者认为FCN把FC1改成卷积层使用的7x7卷积核过大（FCN这么做是为了获得更大的感受域），造成计算量和参数量增大，成为FCN网络的瓶颈。因此本文的方法是对FC1改为了3x3的卷积核，提高了计算速度。

​ 作者也尝试融合了多尺度信息，将输入图像和前四个池化层的输出通过MLP层（MLP层为两层卷积结构。第一个卷积层128 3x3，第二卷积层128 1x1），然后将其输出和最后一个softmax层concat（使softmax输入增加了5*128=640个通道）。MLP是在DCNN训练完成后额外训练的。作者发现多尺度信息能提升细节信息，但是没有CRF来的明显。

4. 实验

本文使用PASCAL VOC 2012数据集。

DeepLab就是在VGG16上改造得来的最普通的DCNN（没有空洞算法），得到59.80%的mIoU（FCN-32s的mIoU为59.4%）；在DeepLab上使用CRF得到DeepLab-CRF模型，获得63.74%的mIoU，得到了4%的提升，这对于CRF已经算很好的成绩（对比其他用CRF的模型只提高了1.5%）；

DeepLab-MSc是使用MLP层融合多尺度特征的模型，相比DeepLab提高1.5%，不如DeepLab结合CRF的提高明显；DeepLab-MSc-CRF是即融合多尺度又使用CRF的模型，相比DeepLab-MSc提高4%，得到65.21%的mIoU。

DeepLab-CRF-7x7是在VGG上直接使用空洞算法（kernel size = 7×7 and input stride = 4）的版本，得到67.64%的mIoU；7x7卷积核是十分

DeepLab-CRF-LargeFOV是使用了kernel size 3×3 和 input stride = 12的版本，与DeepLab-CRF-7x7达到相同的性能，然而参数从134.3M减小到20.5M，运行速度也快了3.36倍。

（这篇论文的实验对比非常丰富，而且deeplab具体的结构是什么也没说，等我有空了仔细研究。fcn的结构也没记住很难弄。迟点来填坑，把网络结构彻底弄清楚）

5. 总结

​ deeplabv1提出了空洞算法，提出了将DCNNs与CRF结合的模型，取得了比FCN好的效果。

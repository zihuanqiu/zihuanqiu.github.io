semseg语义分割库介绍及实现SPNet

本文最后更新于：2020年11月15日 下午

semseg是PSPNet作者构建的一个语义分割库，可用于快速建立模型、训练和验证

1. 环境搭建

Linux系统下：

# 创建虚拟环境
conda create -n pytorch-semseg python==3.7
pip install torch==1.0.0
pip install tensorboardX
conda install opencv
conda install pillow
conda install pyaml

# clone源文件
git clone https://github.com/hszhao/semseg.git
# 安装apex
git clone https://github.com/NVIDIA/apex
cd apex
pip install -v --no-cache-dir --global-option="--cpp_ext" --global-option="--cuda_ext" ./
pip install -v --no-cache-dir ./
# 注意：不能直接使用pip install apex来安装

2. 下载相关数据

  1. 下载数据集

    如：ADEChallengeData2016官方下载地址或者我的百度网盘（提取码：68is）

    修改config文件夹下的配置文件中的数据集路径（绝对路径）

  2. 下载预训练模型权重

    可以使用作者公布的网盘下载下载initmodel，然后将这个文件夹放入工程目录即可（另外两个文件：exp是PSPNet的权重存档，list是数据集的文件名）

    注意：SPNet的resent预训练权重和PSPNet不一样，重新下载
